# Front-end-Special-Project
Create an Angular 8 Web Project 
1. List all users.
2. Select a User.
3. Create a User.
4. Update a User.
5. Delete a User.
6. User Registration.
7. Unsuccessful Registration.


##INSTALLATION

Install dependencies: `npm install`

Start application: `npm start`

open browser and browse: `http://localhost:4200`
